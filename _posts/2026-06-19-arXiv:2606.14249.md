## GLM 5

```
[规格] 744B 总参 / 40B 激活 / 256 experts / 80 层 | MLA→DSA + MTP | base 预算 28.5T tok

 ① Pre-training ─── 27T 量级 通用+代码 语料；标准 MLA 稠密注意力
        │              
        ▼
 ② Mid-training ── 渐进扩窗 32K(1T) → 128K(500B) → 200K(50B)
        │              + ~10M issue-PR 数据 + NextLong 长程合成
        ▼
 ③ DSA 续训 ───── warm-up: 冻结 base 只训 lightning indexer（1000 步）
        │          sparse adaptation: 联合微调，仅 20B tok 追平 MLA（vs V3.2 的 943.7B）
        │          收益: 128K 注意力计算 ↓1.5–2×，成本砍半无损
        ▼
 ④ SFT ────────── ≤202752 tok，三种 thinking 模式
        ▼
 ⑤ Reasoning RL ─ 数学/代码 等可验证答案；GRPO + IcePop
        ▼
 ⑥ Agentic RL ─── 全异步：rollout 与 training 分置不同 GPU 池
        │          SWE/Terminal/Search/Slide 多任务并行，测试通过=grounded reward
        │          
        ▼
 ⑦ General RL ─── 混合 reward（rule + ORM + GRM）+ human-in-the-loop 风格锚，补通用对齐
        ▼
 ⑧ Cross-Stage Distillation ─ on-policy 蒸馏，防灾难性遗忘
```
