# A2A

A2A 本质就是把每个 Agent 做成一个可注册、可发现、可路由、可远程调用的标准服务。

## agent_registy

 self.agents: Dict[str, AgentCard] 保存agents
 app = create_flask_app(self) 基于flask 
 _start_pruning_thread定期清除过期

```python
from python_a2a.discovery import AgentRegistry, run_registry

def start_registry():
    registry = AgentRegistry(
        name="Agent Registry Server", 
        description="Centralized Agent Registry Server"
    )
    run_registry(registry, host="0.0.0.0", port=9100, debug=False)

```

本质 和微服务注册发现几乎一模一样，只是注册内容变成了智能体元数据。

## Agent

A2A 把“agent 的内部实现”隐藏掉，只暴露统一消息接口。
AgentCard: agent 的名片 / 能力广告 / 路由元数据。

```python
class SubAgent01(A2AServer):
    def __init__(self):
        agent_card = AgentCard(
            name="Sub Agent 01",
            description=(
                "Technical specialist agent..."
            ),
            version="1.0.0",
            url="http://127.0.0.1:8001",
            capabilities={
                "reasoning": True,
                "text_generation": True,
                "python": True,
                "code_analysis": True,
                "technical_support": True,
                "google_a2a_compatible": True
            }
        )
        super().__init__(agent_card=agent_card)

    def handle_message(self, message: Message) -> Message:
        prompt = message.content.text
        response = llm.invoke(prompt)
        return Message(
            content=TextContent(text=f"Sub Agent 01: {response.content}"),
            role=MessageRole.AGENT,
            parent_message_id=message.message_id,
            conversation_id=message.conversation_id
        )

    subagent01 = SubAgent01()
    enable_discovery(subagent01, registry_url="http://127.0.0.1:9100")
    run_server(subagent01, host="0.0.0.0", port=8001)
```

## AIAgentRouter DiscoveryClient

本质 llm 根据meta 信息，选用对应的agents。
Router 是 orchestrator 的一个决策部件。
现实需要 LLM semantic routing， 否则 agent 一多，prompt 会越来越长。


```python
client = DiscoveryClient(agent_card=None)
client.add_registry("http://127.0.0.1:9100")

network = AgentNetwork()
for card in client.discover():
    network.add(card.name, card.url)

llm_server = to_a2a_server(create_ollama_llm())
# 智能路由
router = AIAgentRouter(llm_client=llm_server, agent_network=network)

agent_name, confidence = router.route_query("What is Vllm?")
response = network.get_agent(agent_name).ask("What is Vllm?")

```
